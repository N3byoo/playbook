# Playbook

Reusable skills and process docs for building great products with Claude as Senior Agent.

## Structure

- `skills/mobile-app/` — Skills for Expo/React Native iteration cycles
- `processes/vaxmo/` — Project-specific logs, backlog, and decisions for Vaxmo

## How to Use

Reference these skills in CLAUDE.md or invoke them by name in a session.
New projects: copy the relevant skill files and create a new `processes/<project>/` folder.
