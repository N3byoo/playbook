# Vaxmo Playbook Plugin

Reusable skills and processes for Expo/React Native app development with automated iteration cycles.

## Skills

- **expo-iterate** — Core 8-phase iteration loop (Understand → Enrich → Plan → Execute → Verify → Deploy → Retrospective → Report)
- **expo-deploy** — EAS Build deployment for Android APK delivery
- **expo-retrospective** — Post-cycle learnings capture and backlog management

## Commands

- `/iterate <what to improve>` — Run a full iteration cycle

## Usage

These skills trigger automatically based on context. When working on any Expo/React Native app improvement, the expo-iterate skill guides the full process from understanding the request through deployed APK.

## Process Docs

Project-specific tracking lives in `/processes/<project>/`:
- `iteration-log.md` — Running record of each cycle
- `backlog.md` — Prioritized improvement ideas
- `decisions.md` — Key architectural and UX decisions
